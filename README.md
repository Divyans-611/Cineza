# Cineza

## Overview
Cineza is a movie discovery web app where users can explore movies, check details, create watchlists, and share reviews.  
The idea is to make movie discovery more personal, social, and fun by adding AI-based recommendations and gamified user profiles.

## Features
- Search and explore movies in one place
- View movie details like release date, cast, director, genre, rating, and poster
- Check trending, popular, and top-rated movies
- Create a personal watchlist
- Add ratings and reviews for movies
- Build a user profile based on movie taste
- Earn badges, XP, and achievements for movie activity
- Get AI-based movie suggestions according to mood, genre, and preferences
- Clean and responsive UI for mobile and desktop
